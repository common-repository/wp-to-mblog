Tested up to: 3.0.5

支持同步日志的 标题和链接 到各大微博和社区，并且支持使用微博帐号登录 WordPress 博客。

== Description ==

因为某种原因，此插件更名为 WordPress连接微博，请在后台插件页停用该插件，并删除，再安装 WordPress连接微博: http://www.smyx.net/wp-connect.html

以前的设置不变，谢谢支持和谅解！

== Changelog ==

= 1.3.0 =
*2011/02/22
支持使用腾讯微博、新浪微博、网易微博、豆瓣帐号登录WordPress博客，支持同步评论到相应的微博；
非OAuth验证的帐号可以取消绑定了；
当更新日志间隔设为0时，修改日志时将不会再次同步；
同步内容设置新增两个选择；
可以设置禁止同步的日志分类ID，设置后该ID分类下的日志将不会同到微博。
